Если Вы хотите изменить тип страницы, то с данный момент это невозможно сделать через GUI, но так как все страницы хранятся в виде директорий и файлов, то тип страниц можно поменять вручную. Для этого необходимо:

If you want to change the page type, then at given time it is impossible to do through the GUI, but since all pages are stored as files and directories, the page type can be changed manually. To do this:

  1. Открыть директорию страницы, тип которой нужно поменять
  2. Открыть в текстовом редакторе файл __page.opt
  3. В разделе [General] поменять значение type на нужный тип
  4. При необходимости перезагрузить дерево заметок с помощью кнопки Reload wiki

  1. Open the directory page, the type you want to change
  2. Open the file __page.opt in a text editor
  3. In [General] section to change the type value in the wishful type
  4. If you want to restart the tree notes by clickingReload wiki

В данный момент поддерживаются следующие типы страниц:

Currently supported are the following page types:

  • text - /Page Types/Text Page
  • html - /Page Types/HTML Page,
  • wiki - /Page Types/Wiki Page
  • search - /Page Types/Search Page

'Изменять тип страниц надо осторожно, потому что для разных типов файлы __page.opt и __page.text могут использоваться по-разному.''

Change the page type carefully, because for different page types can be used files __page.optand__page.text in different ways.'''
