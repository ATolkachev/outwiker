В каждой заметке можно вставлять ссылки на другую страницу.

You can insert a link to another page in each note.

Примеры ссылок:

Examples of links:

Поиск страницы начинается с корня дерева - /Key Features/Search.

The search of page begins at the root of the tree - /Key Features/Search.

Поиск страницы начинается с текущей записи, но, так как нет заметки с именем "Key Features", снова поиск начинается с корня - Key Features/Search

The search of page begins from the current note, but as "Key Features" note does not exist, search begins from the root again - Key Features/Search

Ссылка на вложенную подстраницу (вложенную заметку): Nested Page

Link to the subpage (subnote): Nested Page

Если ссылка начинается с прямого слеша (/Страница 1/Страница 2…), то поиск такой страницы будет осуществляться в корне дерева. Если начального слеша нет (Страница 1/Страница 2…), то сначала программа будет искать страницу, начиная с текущей страницы, а если не найдет, то опять с корня. Если страница не будет найдена, то ничего не произойдет.

If a link begins with a forward slash (/Page 1/Page 2…), a search of this page begins from the tree root. If a initial forward slash does not exist (Page 1/Page 2…), then the program will search a page beginning from a current page, and if it's not found - then the search beging from the tree root again. If the page is not found - nothing happens.

Wiki-страницы поддерживают свой синтаксис для ссылок.

Wiki pages support its own link syntax.
