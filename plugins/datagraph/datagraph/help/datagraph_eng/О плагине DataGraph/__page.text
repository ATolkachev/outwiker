Плагин [[DataGraph -> http://jenyay.net/Outwiker/DataGraph]] предназначен для построения графиков по текстовым данным. Данные могут вводиться непосредственно на викистранице или читаться из прикрепленных текстовых файлов. 

Для построения графиков используется команда [[[=(:plot:)=] -> page://__e1e1802f-8c19-4e35-8903-a43a1ea57d1d]]. Более подробно о формате данных, с которыми может работать плагин DataGraph читайте в разделе, посвященному команде [[plot -> page://__e1e1802f-8c19-4e35-8903-a43a1ea57d1d/#data]].

Для построения графиков используется библиотека [[Highcharts -> http://www.highcharts.com/]], кроме того, плагин DataGraph использует библиотеку [[python-dateutil -> https://pypi.python.org/pypi/python-dateutil]].
