''Под Linux должны быть установлены следующие пакеты:''

Under Linux should be installed the following packages:

* '''python''' (version 2.6 or 2.7)
* '''python-wxgtk2.8'''
* '''python-wxversion'''
* '''python-webkit-dev'''
* '''libgnomeprint2.2-0''', '''libgnomeprintui2.2-0''' (These packets are used to print text, such as text pages, wiki notation or the original source page. Without this package printing will work, but there is a problem with the stamp of Russian letters)
* '''mimetex''' (without this package will not work rendered formulas)
