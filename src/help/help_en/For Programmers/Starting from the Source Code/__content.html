The program is open source and distributed under the GPL. Sources are located on the launchpad.net.

To get the latest version of the source code from the bazaar repository, execute the command:

bzr branch lp:outwiker/trunk

To run these commands, Bazaar must be installed.

To run the program from the source code, Python 2.6 and wxPython 2.8 must be installed.

To run the OutWiker program from source code, execute following command (in the src directory):

python outwiker.py
