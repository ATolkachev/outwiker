''Исходники программы открыты и распространяются под лицензией GPL. Исходники расположены на [[https://launchpad.net/outwiker| launchpad.net]].''

Programs are open source and distributed under the GPL. Sources located on the [[https://launchpad.net/outwiker | launchpad.net]].

''Чтобы получить самую свежую версию исходников из bazaar-репозитория, выполните команду''

To get the latest version of source from tht bazaar repository, issue the command:

'''bzr branch lp:outwiker/trunk'''

''Для выполнения этих команд должна быть установлена программа [[http://bazaar.canonical.com/en/ | Bazaar]].''

To run these commands must be installed [[http://bazaar.canonical.com/en/ | Bazaar]].

''Для запуска программы из исходников должны быть установлены [[http://www.python.org/ | Python 2.6]] и библиотека [[http://wxpython.org/ | wxPython 2.8]].''

To run the program from the source to be installed [[http://www.python.org/ | Python 2.6]] and [[http://wxpython.org/ | wxPython 2.8]].

''Для запуска программы OutWiker из исходников выполните команду (в директории ''src'')''

To run the program from source OutWiker execute follow command (in the directory''src''):

'''python outwiker.py'''

''При запуске под Linux обратите внимание на [[необходимые пакеты -> /Advanced/Starting under Linux. Depending]]''

When you run on Linux look at [[depending -> / Advanced / Starting under Linux. Depending]]
