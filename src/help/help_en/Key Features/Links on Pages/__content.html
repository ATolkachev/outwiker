В каждой заметке можно вставлять ссылки на другую страницу.

In each note, you can insert a link to another page.

Примеры ссылок:

Examples of links:

Поиск страницы начинается с корня дерева - /Key Features/Search.

The search of page begins at the root of the tree - /Key Features/Search.

Поиск страницы начинается с текущей записи, но так как нет заметки с именем "Key Features", снова поиск начинается с корня - Key Features/Search

The search of page begins at the current note, but as "Key Features" note not exists, again search begins at the root - Key Features/Search

Ссылка на вложенную подстраницу (вложенную заметку): Nested Page

Link to the subnote (the nested note): Nested Page

Если ссылка начинается с прямого слеша (/Страница 1/Страница 2…), то поиск такой страницы будет осуществляться в корня дерева. Если начального слеша нет (Страница 1/Страница 2…), то сначала программа будет искать страницу, начиная с текущей страницы, а если не найдет, то опять с корня. Если страница не будет найдена, то ничего не произойдет.

If a link begins with a forward slash (/Page 1/Page 2…), then a search this page begins at the root. If a initial forward slash not exists (Page 1/Page 2…), then at first program will seak begins at the current page, next again at the root.

Wiki-страницы поддерживают свой синтаксис для ссылок.

Wiki pages support its link syntax.
