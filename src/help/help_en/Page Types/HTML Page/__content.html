Одним из типов страниц, поддерживаемый OutWiker'ом являются страницы, содержащие HTML-код. На данный момент под Windows для показа HTML-страниц используется движок Internet Explorer, а под Linux - WebKit.

One of page types are supported OutWiker, was pages with a HTML code. At the moment for Windows to display HTML pages using Internet Explorer engine, and for Linux - WebKit.

В отличие от стандартного HTML-кода, OutWiker учитывает перевод строк. Так, одиночный перевод строки заменяется на тег <BR>, а двойной перевод строки заменяется на <P>

In contrast to standard HTML code, OutWiker change lines breaks. Thus, a single line break is replaced by the tag <BR>, a double line break is replaced by <P>.

На панель инструментов HTML-страницы вынесены кнопки для добавления некоторых наиболее часто используемых тегов.

The toolbar has buttons for adding some of the most frequently used tags.

Есть возможность добавлять ссылки на другие страницы.

It is possible to add links to other pages.
