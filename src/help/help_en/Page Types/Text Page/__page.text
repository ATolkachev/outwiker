Текстовые страницы - это самый простой тип страниц, поддерживаемый OutWiker'ом. Этот тип страниц не позволяет добавлять никакого оформления  текста или картинки, однако, как и все другие страницы, может также хранить прикрепленные файлы.

Text pages - this is the simplest type of page that is supported by OutWiker. This page type can not add any text formatting or pictures, but, like all other pages can also store attachments.
