Исходники программы открыты и распространяются под лицензией GPL. Исходники расположены на launchpad.net.

Чтобы получить самую свежую версию исходников из bazaar-репозитория, выполните команду

bzr branch lp:outwiker/trunk

Для выполнения этих команд должна быть установлена программа Bazaar.

Для запуска программы из исходников должны быть установлены Python 2.6 и библиотека wxPython 2.8.

Для запуска программы OutWiker из исходников выполните команду (в директории src)

python outwiker.py

При запуске под Linux обратите внимание на необходимые пакеты
